# HeartLink Front-end

## 中文说明

**项目简介**  
面向情侣的前端应用，专注于轻量互动与情绪表达。

**已实现功能**  
- 当前心情实时推送给对方  
- 戳一戳全屏动画  
- AI 恋爱军师（DeepSeek）

**技术栈**  
- Vue 3  
- Vite 5  
- JavaScript  
- Vercel Postgres（项目使用的数据库，后端已与前端分离）

**本地运行**
```bash
npm install
npm run dev
```

**常用脚本**  
- `npm run dev` 本地开发  
- `npm run build` 生产构建  
- `npm run preview` 本地预览构建产物

**说明**  
- 本仓库为前端代码，后端已分离。

---

## English

**Overview**  
A couple-focused front-end app built for lightweight interaction and mood sharing.

**Features Implemented**  
- Real-time mood sync between partners  
- “Poke” full-screen animation  
- AI relationship advisor (DeepSeek)

**Tech Stack**  
- Vue 3  
- Vite 5  
- JavaScript  
- Vercel Postgres (database used by the project; backend is separated)

**Local Development**
```bash
npm install
npm run dev
```

**Scripts**  
- `npm run dev` start dev server  
- `npm run build` production build  
- `npm run preview` preview build locally

**Notes**  
- This repository contains the front-end only. The backend is separated.
